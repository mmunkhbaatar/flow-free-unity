# Flow Free — Unity Clone

[Flow Free](https://play.google.com/store/apps/details?id=com.bigduckgames.flow) тоглоомын
хамгийн энгийн, ажиллагаатай хувилбар (даалгаврын **Option 1**).

## Ажиллуулах заавар

1. Unity Hub → **Add** → энэ хавтсыг сонгоно (Unity **2022.3 LTS** буюу түүнээс шинэ хувилбар).
2. Төсөл нээгдсэний дараа `Assets/Scenes/Main.unity` scene-г нээнэ.
3. **Play** дарна.

> Тоглоомын бүх объект (камер, самбар, UI) runtime дээр кодоор үүсдэг тул
> scene хоосон байгаа нь зөв. Ямар ч scene дээр Play дарахад ажиллана.

## Тоглох арга

- Ижил өнгийн хоёр цэгийг хулганаар чирж холбоно.
- **5 үе** байгаа, бүх өнгөө холбоход үе дуусна.

## Даалгаврын шаардлагууд ба хэрэгжилт

| Шаардлага | Хэрэгжилт |
|---|---|
| Эхний 5 үе | 5 үе бий, бүгд бүрэн шийдэлтэй (самбарыг 100% дүүргэдэг шийдээс зохиогдсон) |
| Зураасаа засах | Зурсан зураасын дундаас дарж тэндээс үргэлжлүүлэн зурж болно; зурж байхдаа буцаж алхвал (backtrack) сүүлийн нүд арилна |
| Өөр зураастай давхцахад давхцсан зураас арилах | Өөр өнгийн зам дээгүүр гарахад тухайн зам давхцсан цэгээсээ хойш тайрагдана |
| Холбосон цэг дээр дарж дахин эхлэх | Endpoint дээр дарахад тухайн өнгийн хуучин зураас бүрэн арилаад шинээр зурж эхэлнэ |
| Үе хожсоны дараа **Next** товч | Бүх өнгө холбогдоход «Дараах (Next)» товч гарна; сүүлийн үе дээр «Эхнээс (Replay)» болно |
| **Replay/Clear** товч | «Цэвэрлэх (Clear)» товч тоглолтын үед үргэлж харагдана, үеийг шинээр эхлүүлнэ |

## Бүтэц

- [`Assets/Scripts/GameManager.cs`](Assets/Scripts/GameManager.cs) — тоглоомын бүх логик нэг файлд:
  - `Bootstrap` — Play дарахад GameManager-ийг автоматаар үүсгэнэ
  - Үеийн өгөгдөл (өнгө бүрийн endpoint-ууд), зурах/засах/тайрах логик
  - SpriteRenderer-ээр дүрслэл, IMGUI-гээр товч/текст

## Дүрэм (Flow Free-тэй адил)

- Зам үргэлж endpoint-ээс эхэлнэ, зөвхөн хөрш нүд рүү сунана.
- Өөр өнгийн endpoint дээгүүр гарч болохгүй.
- Өөрийн зам дээгүүрээ давхцвал тэр цэг хүртлээ тайрагдана.
- Хялбарчилсан ялалтын нөхцөл: бүх хос холбогдсон байхад л хангалттай
  (самбарыг заавал 100% дүүргэх шаардлагагүй).
