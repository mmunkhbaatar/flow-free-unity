# Flow Free clone (Unity)

Даалгаврын Option 1. Unity 2022.3 LTS дээр хийсэн.

## Ажиллуулах

Unity Hub -> Add -> энэ хавтсыг сонгоно. `Assets/Scenes/Main.unity`-г нээгээд Play дарна.

Бүх объект (камер, самбар, товчнууд) кодоос үүсдэг болохоор scene хоосон харагдана,
энэ нь зөв. Код бүгд `Assets/Scripts/GameManager.cs` дотор байгаа.

## Хийсэн зүйлс

- 5 үе, 5x5 хүснэгт. Ижил өнгийн 2 цэгийг чирж холбоно.
- Зураасаа засаж болно: дундаас нь дараад цааш нь зурна, зурж байхдаа ухарвал арилна.
- Өөр өнгийн зураастай давхцвал тэр зураас давхцсан газраасаа тайрагдана.
- Цэг дээр дахиад дарвал хуучин зураас нь арилаад шинээр эхэлнэ.
- Бүх өнгө холбогдоход "Дараах (Next)" товч гарна. "Цэвэрлэх (Clear)" товч байнга байгаа.

Ялалт: бүх хосыг холбоход л хангалттай гэж үзсэн (100% дүүргэх албагүй).
Үе болгон бүтэн дүүргэдэг шийдтэй тул гарцаагүй гарна.
